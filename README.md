# Orrery App

## Building
Instale as dependencias:
```bash
npm install
```

para executar o aplicativo:
```bash
cd src
npx vite
```
e abra o navegador no site mostrado\
Voce tambem pode executar um dos scripts que faz isso automaticamente:
```bash
./launch.sh # para linux
launch.bat # para windows
```
